# 🚀 Asteroid Intelligence System

A full-stack web application that fetches real asteroid data from NASA's NeoWs API and transforms it into interactive visual insights through charts, 3D Earth visualization, intelligent summaries, and downloadable reports.

## 🌟 Features

* Real NASA Asteroid Data Integration
* Interactive Dashboard
* Advanced Search & Filtering
* Hazardous Asteroid Detection
* Dynamic Charts and Analytics
* 3D Earth Visualization using Three.js
* Intelligent Asteroid Insights
* PDF Report Generation
* Voice Input Support
* Modern Glassmorphism UI

## 🛠 Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Libraries

* Chart.js
* Three.js
* jsPDF

### APIs

* NASA NeoWs API

## 📊 Project Workflow

1. User selects a date range.
2. Frontend sends request to backend.
3. Backend fetches asteroid data from NASA API.
4. Data is processed and analyzed.
5. Dashboard displays:

   * Charts
   * Statistics
   * Asteroid Table
   * 3D Earth Visualization
   * Intelligent Insights
